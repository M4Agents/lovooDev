# Sistema de Automação - LovooCRM

## Visão Geral
Sistema de automação visual com 17 ações e 11 triggers.

## Arquitetura

### Componentes Principais
- FlowEditor: Editor visual
- AutomationEngine: Motor de execução
- TriggerManager: Gerenciador de gatilhos
- NodeConfigPanel: Configuração

### Serviços
- automationApi.ts
- CRMService.ts
- NotificationService.ts
- ActivityService.ts
